# IoT-Based Smart WiFi Analyzer System

A full-stack project for monitoring and visualizing WiFi signal quality using an IoT dashboard.

## Overview

- **Frontend:** React + Vite dashboard for displaying WiFi signal strength, network stability, channel distribution, and best network recommendations.
- **Backend:** Express server providing API endpoints for WiFi data and analytics.
- **Data:** Includes sample payload data and backend payload support files.

## Project Structure

- `frontend/` — React application built with Vite.
- `backend/` — Node.js Express server.
- `backend_data.txt` — sample backend data.
- `sample_payload.json` — sample network payload.
- `tmp_frontend_response.html` — temporary frontend output example.

## Installation

1. Install backend dependencies:
   ```bash
   cd backend
   npm install
   ```

2. Install frontend dependencies:
   ```bash
   cd ../frontend
   npm install
   ```

## Running the Project

- Start the backend:
  ```bash
  cd backend
  npm run dev
  ```

- Start the frontend:
  ```bash
  cd frontend
  npm run dev
  ```
